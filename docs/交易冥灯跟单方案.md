# 交易冥灯跟单方案 updated Feb 14

## 一、筛选指标（暂时按照标黄指标初筛）

### 1. 基础指标：筛选有效亏损地址

**目的：** 排除偶尔倒霉的选手和不活跃地址。

以下指标需**同时满足**。

| 指标维度 | 指标 | 量化标准 | 说明 |
|----------|------|----------|------|
| 有效亏损 | 总盈亏 | Total PnL ≤ -$5,000 | 确保地址有真实的亏损 |
| | 胜率 | Win Rate < 40% | 排除运气成分，锁定持续亏损者 |
| | 账户体量 | 账户资金 < $20,000 | |
| | 盈亏比 | PF < 0.5 | PF = 总盈利/总亏损 |
| 地址有效性 | 交易频率 | 过去30天交易（已完成订单）笔数 ≥ 3 | 确保是活跃选手 |
| | 生命周期 | 最早一笔交易距今天数 > 14天 | 排除注册几天就爆仓的 |
| 币种 | 交易对限制 | 交易过Hyperliquid交易额Top10且交易额占80%以上 | |

### 2. 进阶指标：筛选适合反向跟单的地址

**目的：** 识别具备"错误路径依赖"的优质冥灯，为反向跟单服务。

| 目的 | 指标 | 量化标准 | 说明 |
|------|------|----------|------|
| 确定地址价值（需同时满足） | 持仓周转率 | 日均交易笔数 < 20笔 | 排除高频噪音 |
| | 平均持仓时长 | 仓位从open到close的平均时长 > 1h | 确保Hyperbot能成反向对冲 |
| | 大额头寸比 | 历史平均单笔持仓价值 > $10,000 | 只有足够大的仓位才有流动性 |
| 确定跟单价值（Or逻辑，满足项越多越好）——观察指标 | 平均持仓倾斜 | 亏损仓位平均持仓时间 / 盈利仓位平均持仓时间 > 1.5 | 长期亏损的根源："亏损死扛，盈利快跑" |
| | 爆仓依赖度 | 强制平仓金额/总亏损 > 20% | 这类地址不带止损，提供了极好的反向信号 |
| | 爆仓密度 | 过去60天内的爆仓总次数 > 0次 | 频繁爆仓说明风控差，仓价就是你的止盈目标 |
| | 情绪追随(FOMO)系数 | 暂无 | 分子中 开仓/平仓在价格天花板、卖在地板 |
| | 杠杆率 | 历史平均杠杆 > 5x | 过低没爆发力，考虑性价比，10-25x最佳 |

## 二、目标地址 & 资金配置

反向跟单的目标地址最终确认为389个，气泡图形式见 https://bigtomato-cloud.github.io/hyperliquid-trading-lamp/

地址共分为四个象限，按照不同象限对应不同的总体资金配比，具体可见上图。如Q1第一象限，配置总体资金的60%，再平均分配给该象限的100个地址。

- **Q1: 黄金提款机** — 地址数量: 100，建议配置比例: **60%**。低PF + 波动偏高，号码最精确。因为低PF说明它们确在大趋势中长期做反灯，是反向跟单最稳健的利润基石。
- **Q2: 爆破敢死队** — 地址数量: 235，建议配置比例: **25%**。高PF + 波动偏高，号码不稳但爆发强。高PF率意味着偶尔能赚些票但亏更凶，适合博弈快速金的利润。
- **Q3: 鸡肋观察区** — 地址数量: 11，建议配置比例: **10%**。低PF + 平稳偏低，号码质朴但不太炸。偶尔能翻盘运气尚且可以，由于仓位低且PF偏高，反向跟单的资金占用效价比偏低。
- **Q4: 高危禁区** — 地址数量: 43，建议配置比例: **5%**。高PF + 高明显度，信号虽远但存在巨大的翻转风险。高PF条表示偶尔胜利不少又会连续亏损，也希望通过止损反方向获利暴跌通道。
