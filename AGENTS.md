## TDD

- 代码变动 → 默认跑 `npm test`（全量单测，不含外部 API 集成测试）
- 接口相关测试（`src/lib/__tests__/hyperbot.api.test.ts`）仅在以下场景必跑：改动 `src/lib/hyperbot.ts`/该测试文件、改动 API 调用链、或用户明确要求。命令：`npm run test:api`
- 覆盖每行代码，ROI 不值得需注释说明
